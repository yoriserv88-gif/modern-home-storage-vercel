# 横幅管理器使用指南

## 功能说明
这是一个完整的横幅管理后台，让你可以在线上传和管理首页横幅图片，无需修改代码。

## 访问路径
```
http://localhost:3000/admin/banners
```

## 主要功能

### 1. 查看所有横幅
- 显示当前所有横幅的预览
- 显示横幅的标题、副标题、CTA 按钮文字
- 显示背景颜色

### 2. 上传图片
- 点击 "Upload Image" 按钮
- 选择本地图片文件（JPG, PNG, WebP）
- 修改横幅标题、副标题、CTA 文字（可选）
- 选择背景颜色
- 点击 "Upload" 完成上传

### 3. 调整顺序
- 使用 ↑ ↓ 按钮调整横幅显示顺序
- 上面的横幅优先显示

### 4. 编辑横幅
- 点击 "Upload Image" 重新上传图片
- 修改横幅的文字内容

### 5. 删除横幅
- 点击删除按钮
- 确认后删除横幅

## 数据存储
- 横幅配置保存在：`data/banners.json`
- 上传的图片保存在：`public/images/banners/`

## 使用流程

### 上传新图片替换旧横幅

1. 访问 `/admin/banners`
2. 找到你想替换的横幅（比如第 1 张）
3. 点击该横幅的 "Upload Image" 按钮
4. 选择你上传的图片文件（比如 `hero-banner.jpg`）
5. 确认修改
6. 完成！新图片会自动替换旧图片

### 添加新的横幅

1. 点击右上角 "Add New Banner" 按钮
2. 填写横幅信息
3. 选择图片
4. 点击 "Upload"

## 注意事项

1. **图片尺寸**：建议使用 1920x1080px 的横幅图片
2. **文件大小**：建议不超过 2MB
3. **文件格式**：支持 JPG, PNG, WebP
4. **图片命名**：系统会自动重命名为 `banner_{id}.jpg`

## 和之前方法的区别

| 方法 | 优点 | 缺点 |
|------|------|------|
| 修改 `data/banners.ts` | 简单直接 | 需要重启服务，需要代码访问权限 |
| **Banner Admin (新)** | 在线管理，无需重启，无需代码权限 | 需要登录后台 |

## 安全性
- 需要管理员登录（admin_auth cookie）
- 限制图片类型
- 限制文件大小（可在 API 中添加）
